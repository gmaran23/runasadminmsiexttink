Using RunAsAdminMSIExtTinkerer.exe is as simple as

  1. Run the executable named RunAsAdminMSIExtTinkerer.exe as Administrator
  2. Tick the check box to enable Run as Administrator for msi files
  3. Clear the Tick in the check box to disable Run as Administrator for msi files

There is a user guide with screenshots attached to the redistributable package on te Downloads section.

RunAsAdminMSIExtTinkererUserManual.pdf is also available as a direct download from the Downloads section.
